✊✋✌️ Rock-Paper-Scissors Game

Project Description

The Rock-Paper-Scissors Game is a fun interactive web application built using HTML, CSS, and JavaScript.

It allows users to play against the computer by selecting Rock, Paper, or Scissors. The computer makes a random choice and the winner is decided instantly.

This project demonstrates the use of DOM manipulation, event handling, and basic game logic in JavaScript.

Key Features

  • 🎮 Interactive gameplay against the computer
  • ⚡ Instant winner result display
  • 🎨 Simple and user-friendly design
  • 📱 Responsive for all devices
