📒 Address Book Application

Project Description

The Address Book application is a web-based tool designed to manage personal or professional contacts efficiently. Built with HTML, CSS, and JavaScript, it allows users to add, view, edit, delete, and search contacts seamlessly within a browser. It’s a beginner-friendly project to learn core front-end concepts and DOM manipulation.

Key Features

  • ✅ Add New Contacts
  • 👀 View Contact List
  • ✏️ Edit Existing Contacts
  • 🗑️ Delete Contacts
  • 🔍 Search Functionality
  • 📱 Responsive Design
