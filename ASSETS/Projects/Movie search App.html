🎬 Movie Search Application

Project Description

The Movie Search Application is built using HTML, CSS, and JavaScript. It allows users to search for movies and instantly fetch details like title, release year, genre, and poster using a movie API.

This project enhances user experience with a clean interface, real-time results, and interactive design.

Key Features

  • 🔍 Search movies by title
  • 🎥 Display posters, release year, and genre
  • 📱 Responsive and user-friendly design
  • ⚡ Real-time results using API
